Instagram投稿を自動化

画像を自動でトリミングし、AIが魅力的な投稿文とハッシュタグを生成します

⚠️ これはデモページです

実際の動作にはNode.jsサーバーの起動が必要です。以下のセットアップ手順をご確認ください。

🚀 セットアップ方法 ✨ 機能を見る

主な機能

📷

自動画像処理

アップロードした画像を自動で1:1の正方形にトリミング。 Instagram推奨の1080x1080pxに最適化します。

🤖

AI文章生成

Gemini APIを使用して、画像に最適な投稿文を自動生成。 「まじめ」「ユーモア」「キラキラ」の3つのスタイルから選択可能。

#️⃣

ハッシュタグ生成

関連性の高いハッシュタグを自動生成。 「多い」「普通」「少ない」から量を選択できます。

セットアップ手順

Terminal 
# 1. リポジトリをクローン
git clone https://github.com/tsukasa19910921/instagram-helper.git
cd instagram-helper

# 2. 依存関係をインストール
npm install

# 3. 環境変数を設定
cp .env.example .env
# .envファイルを開いてGemini APIキーを設定

# 4. サーバーを起動
npm start

# 5. ブラウザでアクセス
# http://localhost:3000

Gemini APIキーが必要です

🔑 APIキーを取得する（無料）
